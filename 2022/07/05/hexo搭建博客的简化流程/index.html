hexo搭建博客的简化流程

发表于2022-07-05
|字数总计:1.1k|阅读时长:4分钟|阅读量:

环境配置

安装node.js

  • hexo是基于node框架的,所以需要先安装node环境
  • node下载地址: node.js
  • 安装完成后打开cmd输入node -v检查是否安装成功,显示版本号即为安装成功

安装 git

​ 对于没有自己服务器的小伙伴,部署到免费的github pages无疑是最好的选择,那分布式版本控制系统git也将会是必要的工具

​ 下载地址: git

​ 下载完成后可以在cmd中输入 git --version 来检查是否安装成功

安装hexo并初始化博客

  1. 打开cmd输入 npm install hexo-cli -g 来安装hexo

  2. 输入hexo init xxx.github.io 初始化本地博客目录

    xxx换成你喜欢的名字

  3. cd xxx.github.io 进入目录

  4. 安装npm install

  5. 清除缓存 hexo clean 或者 hexo cl

  6. 生成本地的静态页面 hexo g 或者 hexo generate

  7. 运行网站 hexo s 或者 hexo serve

    5 6 7步可以简写为 hexo cl && hexo g && hexo s

  8. 打开浏览器输入 localhost:4000,就可以看到新建的默认博客啦~

优化主题

为了方便快速高效的美化你的博客,hexo官网提供了300+个主题供用户选择,并且每个主题都有自己的说明文档供用户自由修改

本文以butterfly主题为例演示如何快速配置自己喜欢的主题以及部分简单样式

相关链接:

  • hexo
  • hexo_theme
  • butterfly主题

下载

  1. 在自己的博客目录下 cmd中输入 git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly
  2. 下载完成后可以在资源管理器中看到themes文件夹中多出了一个butterfly的文件夹,说明主题下载完成

配置主题

下载完成后 在博客根目录里找到 _config.yml 文件,修改theme为你的主题的名字

重新清除缓存,生成,运行 : hexo cl && hexo g && hexo s

打开localhost:4000 就可以看到butterfly的默认界面啦
